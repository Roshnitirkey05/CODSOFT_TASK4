Rock Paper Scissors – Python Tkinter Game

A modern Rock Paper Scissors desktop game built using Python and Tkinter.
The game features an interactive UI, score tracking, round counter, and smooth animations that make the gameplay engaging and fun.

Features

- 🪨 Rock, Paper, Scissors gameplay
- 🤖 Random computer opponent
- 📊 Live score tracking
- 🔁 Round counter
- 🎨 Modern dark UI design
- ⚡ Computer thinking animation
- 🔄 Reset game option
- ✨ Hover effects on buttons

 Built With

- Python
- Tkinter (GUI library)
- Random module

How the Game Works

1. The player selects Rock, Paper, or Scissors.
2. The computer randomly chooses one option.
3. The program compares both choices and determines the winner.
4. Scores and rounds are updated automatically.

Game Rules:

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
